## Installation

### Requeriment

- Node >= 12
- Postgres >= 10

### Clone

- Clone this repo to your local machine using `https://github.com/filafb/favelainc.git`

### Setup

> install project dependencies

```shell
npm install
```

> Run webpack and start dev server

```shell
npm run start:dev
```
